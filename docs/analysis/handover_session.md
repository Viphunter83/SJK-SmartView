# План на следующую сессию (Handover) — SJK SmartView

Данный документ предназначен для быстрого восстановления контекста и продолжения разработки в новом чате.

## 1. Текущий статус проекта
*   **Стадия:** Завершена Фаза 1 (Фундамент и Каталог) и Фаза 2.1 (Контейнеризация).
*   **Stack:** Next.js 15 (Frontend), FastAPI Python 3.12 (Backend), PostgreSQL 16, Docker.
*   **Инфраструктура:** Проект полностью контейнеризирован. Запуск через `docker-compose up --build`.

---

## 2. Что уже реализовано
*   **Frontend:**
    *   Сайдбар, Каталог объектов с поиском.
    *   Модальное окно `MockupCreator` с интеграцией реального API.
*   **Backend:**
    *   Эндпоинт `/v1/mockup/generate` для приема изображений.
    *   Скелет GPU-воркера для Modal.com (`worker.py`).
    *   Схема БД и миграции (locations, users, mockups).
*   **Deployment:**
    *   Конфиги для Railway (`railway.json`) и Vercel.
    *   Git репозиторий готов к пушу.

---

## 3. С чего начать в следующей сессии (Backlog)

### Приоритет 1: AI Пайплайн (Реализация воркера)
*   **Задача:** Оживить `worker.py`. Подключить реальный **SAM 2/3** для сегментации.
*   **Действие:** Настроить Modal-токены в `.env` и провести первый запуск удаленной функции.
*   **Результат:** Фотореалистичное наложение баннера под провода/деревья.

### Приоритет 2: Реальные данные Shojiki
*   **Задача:** Наполнить базу данных реальными экранами из Вьетнама.
*   **Действие:** Написать скрипт-парсер или миграцию для добавления 10-15 топовых локаций Shojiki с их координатами и масками.

### Приоритет 3: PWA и Камера
*   **Задача:** Реализовать режим «Street Upload».
*   **Действие:** Добавить во фронтенд доступ к камере телефона и логику загрузки фото «с поля» для обработки по AI-пайплайну.

## 5. Команды для быстрого старта
```bash
# Запуск всей инфраструктуры
docker-compose up -d

# Локальный запуск бэкенда (если не через Docker)
cd backend && pip install -r requirements.txt && uvicorn app.main:app --reload

# Локальный запуск фронтенда
cd frontend && npm install && npm run dev
```
