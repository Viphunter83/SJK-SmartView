# Handover: SJK SmartView (Vietnam Edition)

Этот документ фиксирует состояние проекта после стабилизации UI, внедрения авторизации и перехода на реальные данные Shojiki Group во Вьетнаме.

## 1. Текущий статус проекта
*   **Стадия:** Завершена Фаза 2 (Стабилизация и Интеграция). Система готова к работе менеджеров в "ручном" режиме (с использованием пресетов).
*   **Локализация:** База данных наполнена реальными экранами Хошимина и Ханоя (Shojiki.vn).
*   **Инфраструктура:** Docker-окружение стабильно. Подключены Firebase Auth и облачное хранилище Firebase Storage.

## 2. Что реализовано в этой сессии
*   **Frontend:**
    *   **Авторизация:** Реализован `AuthContext` и защищенные маршруты. Вход через Firebase.
    *   **UI Stability:** Удалены конфликты провайдеров, исправлены ошибки гидратации в `MockupCreator`.
    *   **Каталог:** Корректное отображение вьетнамских локаций с обработкой абсолютных URL и fallback-изображениями.
*   **Backend:**
    *   **Firebase Admin:** Интегрирован `service_account.json` для работы со Storage.
    *   **Database:** Выполнено полное сидирование (locations, users) для вьетнамского рынка.
    *   **StorageService:** Теперь файлы загружаются напрямую в Firebase Storage (bucket: `sjk-smartview.firebasestorage.app`).

## 3. Задачи для следующей сессии (Backlog)

### Приоритет 1: AI Worker & Segmentation (Modal.com)
*   **Задача:** Реализовать реальный пайплайн обработки.
*   **Действие:** Развернуть `worker.py` на Modal. Интегрировать **SAM 3** для автоматического вырезания проводов/деревьев перед экраном.
*   **Результат:** Фотореалистичные мокапы без ручного редактирования.

### Приоритет 2: Автоматическая детекция углов
*   **Задача:** Оживить кнопку "Auto-detect".
*   **Действие:** Обучить/подключить легковесную модель YOLOv12-OBB для поиска рамок экранов на произвольных фото ("с улицы").

### Приоритет 3: PWA и Мобильная оптимизация
*   **Задача:** Удобство работы "в полях".
*   **Действие:** Добавить manifest.json, настроить кэширование и доступ к камере для режима "Street Upload".
