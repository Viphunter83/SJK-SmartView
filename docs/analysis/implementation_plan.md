# Финальный план реализации SJK SmartView

## 1. Концепция и Технологический стек
SJK SmartView — корпоративное PWA для быстрой визуализации DOOH-рекламы.

*   **Frontend:** Next.js 15, TailwindCSS, Shadcn UI.
*   **PWA:** Прямой доступ к камере, Service Workers для офлайн-каталога.
*   **Backend:** FastAPI (Python) — роль диспетчера и легкой обработки.
*   **AI (Modern 2026):**
    *   **Детекция:** YOLOv12-OBB (Oriented Bounding Boxes).
    *   **Сегментация:** SAM 3 (Segment Anything Model) для масок окклюзий.
*   **GPU Инфраструктура:** Serverless GPU (Modal / Replicate) для минимизации затрат.
*   **DB:** PostgreSQL (Supabase/Railway).

---

## 2. Ключевые архитектурные решения

### ⚡️ Экономика и Скорость (Hybrid AI)
1.  **Каталог экранов:** Для существующих в базе объектов маски (окклюзии) и параметры перспективы **предрассчитаны**. Наложение происходит мгновенно на клиенте через Canvas/WebGL. Серверный GPU не используется.
2.  **Фото с улицы:** Новые фото обрабатываются через диспетчер.
    *   **Fast Path:** Классическая цветокоррекция и наложение (OpenCV) — 1 секунда.
    *   **Deep Path (Опционально):** Диффузионный Relighting (ControlNet) — 5-10 секунд для "премиальных" видов (ночь/сложный свет).

### 📱 Клиентская оптимизация
*   **Web Workers:** Ресайз тяжелых фото (до 2048px) прямо в браузере перед отправкой. Экономия трафика менеджера.
*   **Progressive Loading:** Отображение "скелетонов" в то время, когда AI строит маску.
